# Software Licensing and Subscription Requests

**Category:** Purchase  

## Description
Requests for new or additional software licenses and SaaS subscriptions.

## Typical Ticket Patterns
- “Need new Adobe license”
- “Additional Jira seat required”
- “Renew SaaS subscription”

## Resolution Steps
1. Confirm software is approved and supported by IT.
2. Check existing unused licenses or subscriptions before purchasing new ones.
3. Validate business justification and number of seats required.
4. Raise purchase or allocation request and update license inventory.
5. Provide installation or access instructions once license is assigned.

## When to Escalate
- New SaaS tools not yet vetted by security/legal.
- Enterprise-level contracts or renewals.
- License shortfalls impacting compliance.
