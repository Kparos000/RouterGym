# HR System Access Guide (HRIS / Oracle)

**Category:** HR Support  

## Description
Granting and managing access to core HR systems (e.g., HRIS, Oracle, SAP HR).

## Typical Ticket Patterns
- “Cannot log in to HR system”
- “Need HR reporting role”
- “New manager cannot approve leave in Oracle”

## Resolution Steps
1. Confirm user's employment status, role, and manager from HR.
2. Map requested rights to standard HR role profiles (employee, manager, HR staff).
3. Submit or approve access change in HR system admin tool.
4. Ask user to log out of SSO and back in to refresh permissions.
5. Validate that key tasks (time entry, leave approval, HR reporting) now work.
6. Update ticket with final access profile and any exceptions.

## When to Escalate
- Requests for custom or non-standard HR roles.
- Conflicting instructions from HR and manager.
- HR system incidents or performance issues affecting many users.
