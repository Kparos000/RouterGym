# Restricted Area or Building Access Requests

**Category:** HR Support  

## Description
Requests for card access to specific buildings, zones, or secure rooms.

## Typical Ticket Patterns
- “Badge not working at new office”
- “Need access to secure lab”
- “Turnstile denies entry”

## Resolution Steps
1. Confirm role, manager approval, and reason for access.
2. Check user status in physical access control system.
3. Assign appropriate access profile and time schedule.
4. Ask user to test badge at required entry points.
5. Log access level changes for audit.

## When to Escalate
- Security incidents or lost/stolen badges.
- Access to high-risk areas (data centres, labs).
- Conflicts between requested access and security policy.
